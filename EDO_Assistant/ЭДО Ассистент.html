Руководство пользователя для программы ЭДО Ассистент

Это руководство поможет вам настроить и использовать программу ЭДО Ассистент для автоматизации загрузки файлов через браузер.

Оглавление

  • Конфигурационный файл
  • Интеграция с xprint.PLUS

Конфигурационный файл

Конфигурационный файл config.txt содержит настройки, необходимые для работы программы. Вот описание параметров:

  • chromePath — путь к исполняемому файлу браузера Chrome (необязательно, по умолчанию используется Edge).
  • userDataDir — путь к папке с пользовательскими данными браузера. Это необходимо для использования существующего профиля браузера.
  • uploadRule — правило загрузки файлов. Формат: ИНН_отправителя,ИНН_получателя,путь_к_сайту. Например:
    • uploadRule=1234567890,0987654321,https://example.com/upload

Интеграция с xprint.PLUS

Программа может быть интегрирована с программой **Электронные документы - xprint.PLUS**. Она автоматически запускается при выгрузке документов в формате XML. Вот как это работает:

  1. Настройка xprint.PLUS: Убедитесь, что в настройках xprint.PLUS включена опция ЭДО Ассистент.
  2. Автоматический запуск: При выгрузке документа в XML, программа автоматически запускается и обрабатывает файл. Она использует данные из XML (например, ИНН отправителя и получателя) для определения правил загрузки.
  3. Загрузка файла: Программа загружает файл на указанный в конфигурации URL, используя правила, заданные в config.txt.

Примечание: Для корректной работы интеграции убедитесь, что программа располагается в папке "C:\EDO_Assistant", а также что конфигурационный файл config.txt содержит правильные данные.
